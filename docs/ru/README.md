# 📚 Документация DataCode на русском языке

Добро пожаловать в полную документацию языка программирования **DataCode** (ДатаКод) на русском языке!

## 🎯 О языке DataCode

**DataCode** — простой интерактивный язык программирования, разработанный для быстрой обработки данных и легкого обучения. Он отличается интуитивным синтаксисом, мощной поддержкой массивов, встроенными функциями и пользовательскими функциями с локальной областью видимости. Детали реализации VM и рантайма (глобалы, модули, профилирование) см. в [Ядро / Internals](./internals/README.md).

### 12. [Модули и импорты](./modules_and_imports.md)

Импорт и использование модулей в DataCode:

- **Синтаксис импорта** — `import M`, `from M import X, Y as Z, *`
- **Базовый путь** — задаётся директорией скрипта при запуске из CLI; локальные модули разрешаются относительно него
- **Модуль-файл** — `<имя>.dc`; **пакет** — `<имя>/__lib__.dc` (приоритет над файлом)
- **Составные имена** — `core.config` для вложенных пакетов (каждый сегмент = пакет или файл)
- **Встроенные модули** — `ml`, `plot`, `settings_env`, `uuid`, `database`
- **Экспорты** — глобалы верхнего уровня (переменные, функции, классы) модуля .dc
