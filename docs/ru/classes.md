## Объявление класса

Класс объявляется ключевым словом `cls`. Тело класса заключается в фигурные скобки.

**Синтаксис без наследования:**

```datacode
cls ClassName {
    # поля, конструкторы, методы
}
```

**Синтаксис с наследованием:**

```datacode
cls ChildClass(ParentClass) {
    # поля, конструкторы, методы
}
```

Имя родительского класса в скобках должно ссылаться на ранее определённый класс.

## Абстрактные классы

Класс можно пометить как абстрактный, указав атрибут `@Abstract` непосредственно перед `cls`. Абстрактный класс нельзя инстанцировать напрямую; попытка вызова (например, `AbstractBase()`) приводит к ошибке времени выполнения: `Cannot instantiate abstract class 'AbstractBase'`. Наследники абстрактного класса — обычные классы, их можно инстанцировать. Конструкторы и вызов `super(...)` в наследниках работают как обычно.

**Синтаксис:**

```datacode
@Abstract
cls AbstractBase {
    public:
        name: str
    new AbstractBase(name) {
        this.name = name
    }
}

cls Concrete(AbstractBase) {
    new Concrete(name) {
        super(name)
    }
}

let c = Concrete("ok")   # разрешено
# let b = AbstractBase("x")  # ошибка времени выполнения: Cannot instantiate abstract class 'AbstractBase'
```

## Секции видимости

Члены класса группируются по видимости: `private:`, `protected:` и `public:`.

- **private** — доступ только внутри класса, который определяет член.
- **protected** — доступ в определяющем классе и в наследниках (не снаружи).
- **public** — доступ откуда угодно (поля и методы экземпляра).

Секции необязательны. Члены, объявленные без предшествующей секции, по умолчанию считаются private. Секции можно повторять (например, несколько блоков `public:`).

```datacode
cls Example {
    # секция по умолчанию (private)
    secret: int

    protected:
    internal: str

    public:
    name: str
    new Example(name) {
        this.name = name
    }
}
```

## Поля

Поля объявляются с опциональным типом и опциональным значением по умолчанию.

**Формы:**

- `name: type` — поле с типом, без значения по умолчанию
- `name: type = expression` — поле с типом и значением по умолчанию (константные выражения по возможности вычисляются на этапе компиляции)

Поля перечисляются в секциях `private:`, `protected:` или `public:`.

```datacode
cls Point {
    public:
        x: int
        y: int
    new Point(x, y) {
        this.x = x
        this.y = y
    }
}
```

```datacode
cls Counter {
    private:
        n: int = 0
    public:
        name: str
    new Counter(name) {
        this.name = name
    }
}
```

## Переменные уровня класса

Переменные уровня класса — это присваивания без аннотации типа: `name = expression`. Они хранятся в объекте класса (например, для конфигурации). Подчиняются тем же секциям видимости, что и поля.

```datacode
cls Config {
    public:
        model_config = Settings.config(env_prefix="APP__")
}
```

## Конструктор

Конструктор объявляется как `new ClassName(params)` с телом в фигурных скобках.

**Синтаксис:**

```datacode
new ClassName(param1, param2, ...) {
    # тело: присваивание this.field, вызов super(...) при наследовании и т.д.
}
```

**Создание экземпляра:** вызовите класс как функцию с тем же числом аргументов. Среда выбирает конструктор по количеству аргументов (например, `ClassName::new_2` для двух параметров).

```datacode
let p = Point(10, 20)
let c = Counter("MyCounter")
```

**Делегирующий конструктор:** чтобы вызвать другой конструктор того же класса, после списка параметров укажите `: this(...)` и оставьте тело пустым:

```datacode
new ClassName(a, b) : this(a + b) { }
```

## Наследование

Чтобы унаследовать класс, объявите `cls Child(Parent) { ... }`.

**Конструктор родителя:** в конструкторе класса-наследника нужно ровно один раз вызвать конструктор родителя первой инструкцией: `super(args)`.

```datacode
cls Child(Parent) {
    public:
    new Child(v) {
        super(v, 202)
    }
}
```

**Вызов метода родителя:** внутри метода используйте `super.methodName(args)`. Это допустимо только внутри класса с родителем.

```datacode
fn overridden() {
    return super.overridden() + 1
}
```

Приватные члены родителя недоступны в наследнике. Защищённые (protected) доступны в классе-наследнике (например, в методах); среда выполнения запрещает чтение и запись protected-полей снаружи иерархии классов.

## Методы

Методы объявляются как `fn methodName(params) { body }`. Опционально можно указать тип возвращаемого значения: `fn methodName(params) -> type { body }`. Первый неявный параметр — `this` (текущий экземпляр).

```datacode
cls Calculator {
    public:
        base: int
    new Calculator(base) {
        this.base = base
    }
    fn add(n) {
        return this.base + n
    }
}
```

**Вызов метода:** `instance.methodName(args)`.

```datacode
let calc = Calculator(100)
print(calc.add(25))  # 125
```

## Параметр @class

Метод может объявить необязательный первый параметр `@class`, чтобы получить объект класса (мета-информацию о классе). Он должен быть первым параметром в списке. VM автоматически подставляет объект класса при вызове метода; вызывающий код его не передаёт.

**Синтаксис:** `fn methodName(@class, otherParams) { body }`

**Пример:**

```datacode
cls User {
    public:
        name: str
    new User(name) {
        this.name = name
    }
    fn info(@class) {
        print(@class.name)           # "User"
        print(@class.full_name)      # "User"
        print(@class.method_names)   # ["info", ...]
    }
}

let u = User("Alice")
u.info()   # @class подставляется; выводит имя класса и список методов
```

**API объекта @class (свойства объекта класса):**

- **Идентификация:** `@class.name` (короткое имя), `@class.full_name` (пока совпадает с name)
- **Иерархия:** `@class.parent` (имя суперкласса или null)
- **Модификаторы:** `@class.is_abstract` (true, если класс абстрактный)
- **Структура:** `@class.method_names` (массив имён методов)

Используйте `@class` для регистрации в ORM, сериализации, DI или любой логики, которой нужно знать класс во время выполнения без изменения мест вызова.

## Доступ к полям

- **Внутри класса:** используйте `this.fieldName` для чтения и присваивания полей экземпляра в конструкторах и методах.
- **Снаружи:** используйте `object.fieldName` для публичных полей. Обращение к private или protected полям снаружи класса (или из «неправильного» класса в иерархии) приводит к ошибке времени выполнения (например, ProtectError).

```datacode
# В конструкторе или методе
this.x = 10
let v = this.x

# Снаружи (только public)
let p = Point(5, 10)
print(p.x)
print(p.y)
```
