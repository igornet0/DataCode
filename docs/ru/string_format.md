# Форматирование строк (интерполяция)

В DataCode строки поддерживают **интерполяцию**: подстановку значений переменных и выражений внутрь строки в формате `${...}`. Доступны варианты с выводом имени переменной и с форматированием чисел.

## Базовая интерполяция

Внутри строки в двойных кавычках можно использовать `${выражение}`. Выражение вычисляется, и его значение подставляется в строку (как при преобразовании в строку).

```datacode
let name = "Игорь"
"Привет, ${name}!"   # "Привет, Игорь!"

let a = 2
let b = 3
"результат: ${a + b}"   # "результат: 5"
```

Поддерживаются любые выражения: переменные, вызовы функций, арифметика, доступ к свойствам и т.д.

---

## Вывод «имя=значение»

Синтаксис **`${переменная=}`** выводит не только значение, но и имя в виде `имя=значение`. Удобно для отладочного вывода и логов.

```datacode
let n = 42
let b = true
"${b=} ${n=}"   # "b=true n=42"
```

Для сложных выражений в качестве «имени» используется текст выражения (например, `a+b=5`).

---

## Форматирование чисел

Синтаксис **`${выражение:формат}`** позволяет задать формат вывода числа. Формат задаётся после двоеточия и похож на спецификатор float в стиле Python.

| Формат | Описание        | Пример (n = 42.23425) |
|--------|-----------------|------------------------|
| `.0f`  | Целое (0 знаков после запятой) | `42`   |
| `.2f`  | Два знака после запятой       | `42.23`|
| `.4f`  | Четыре знака после запятой     | `42.2343` |

```datacode
let n = 42.23425
"${n:.2f}"    # "42.23"
"${n:.0f}"    # "42"
```

Для нечисловых значений формат игнорируется, используется обычное строковое представление.

---

## Комбинирование

Можно одновременно выводить имя и применять формат: **`${переменная=:формат}`**.

```datacode
let n = 42.23425
"${n=:.0f}"   # "n=42"
"${n=:.2f}"   # "n=42.23"
```

Порядок: сначала выражение, затем необязательный суффикс `=`, затем необязательный `:формат`.

---

## Экранирование

Чтобы вставить в строку буквальную последовательность `${` (без интерполяции), экранируйте символ доллара: **`\${`**. В результате в строку попадёт `${`.

```datacode
"Синтаксис: \${expr}"   # "Синтаксис: ${expr}"
```

---

## Краткая справка

| Синтаксис           | Результат (пример)      |
|---------------------|-------------------------|
| `${x}`              | значение выражения      |
| `${x=}`             | `x=значение`            |
| `${x:.2f}`          | число с 2 знаками после запятой |
| `${x=:.0f}`         | `x=42` (имя и целое)     |
