# 🚀 Основы ДатаКод

Этот раздел содержит базовые примеры для изучения основ языка программирования ДатаКод.

## 📋 Содержание

### 1. `привет.dc` - Простейший пример
**Описание**: Демонстрирует базовый вывод текста.

**Что изучается**:
- Функция `print()`
- Строковые литералы

**Запуск**:
```bash
datacode examples/ru/01-основы/привет.dc
```

### 2. `переменные.dc` - Переменные
**Описание**: Демонстрирует объявление и использование переменных.

**Что изучается**:
- Объявление переменных с `let`
- Присваивание значений
- Использование переменных в выражениях

**Запуск**:
```bash
datacode examples/ru/01-основы/переменные.dc
```

### 3. `арифметика.dc` - Арифметические операции
**Описание**: Демонстрирует математические операции.

**Что изучается**:
- Арифметические операторы: `+`, `-`, `*`, `/`
- Приоритет операций
- Унарные операторы

**Запуск**:
```bash
datacode examples/ru/01-основы/арифметика.dc
```

### 4. `строки.dc` - Работа со строками
**Описание**: Демонстрирует операции со строками.

**Что изучается**:
- Строковые литералы
- Конкатенация строк (`+`)
- Функция `len()` для получения длины строки

**Запуск**:
```bash
datacode examples/ru/01-основы/строки.dc
```

### 5. `глобальные_локальные.dc` - Глобальные и локальные переменные
**Описание**: Демонстрирует область видимости глобальных и локальных переменных.

**Что изучается**:
- Глобальные переменные с `global`
- Локальные переменные с `let`
- Область видимости переменных в функциях

**Запуск**:
```bash
datacode examples/ru/01-основы/глобальные_локальные.dc
```

### 6. `классы.dc` - Работа с классами
**Описание**: Демонстрирует объявление классов, конструкторов, полей и методов.

**Что изучается**:
- Простой класс с публичными полями и конструктором (`cls Point`, `new Point`)
- Класс с приватными полями и методами (`Counter`, `inc()`, `get()`)
- Класс с методом, принимающим аргументы (`Calculator`, `add(n)`)

**Запуск**:
```bash
datacode examples/ru/01-основы/классы.dc
```

### 7. `наследование.dc` - Наследование классов
**Описание**: Демонстрирует классы-наследники, вызов `super` и доступ к унаследованным и защищённым полям.

**Что изучается**:
- Родительский класс с публичными и защищёнными полями
- Класс-наследник с вызовом `super()` в конструкторе
- Доступ через методы и прямой доступ; `ProtectError` при обращении к защищённым полям

**Запуск**:
```bash
datacode examples/ru/01-основы/наследование.dc
```

## 🎯 Изучаемые концепции

### Переменные
- **Объявление**: `let имя = значение`
- **Типы данных**: числа, строки, булевы значения
- **Автоматическое определение типов**

### Операторы
- **Арифметические**: `+`, `-`, `*`, `/`
- **Строковые**: конкатенация с `+`
- **Унарные**: `-` (отрицание)

### Встроенные функции
- **`print(...)`**: вывод данных на экран
- **`len(str)`**: получение длины строки

## ⚠️ Важные моменты

1. **Объявление переменных**: Используйте `let` для объявления переменных
2. **Типы данных**: ДатаКод автоматически определяет типы
3. **Строки**: Используйте двойные кавычки `"text"`
4. **Комментарии**: Начинаются с символа `#`
