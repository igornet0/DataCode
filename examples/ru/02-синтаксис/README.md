# 🔧 Синтаксис ДатаКод

Этот раздел содержит примеры синтаксических конструкций языка.

## 📋 Содержание

### 1. `условия.dc` - Условные конструкции
**Описание**: Демонстрирует использование `if` и `else`.

**Что изучается**:
- Условная конструкция `if`
- Конструкция `if/else`
- Вложенные условия
- Сравнение значений

**Запуск**:
```bash
datacode examples/ru/02-синтаксис/условия.dc
```

### 2. `выражения.dc` - Сложные выражения
**Описание**: Демонстрирует различные типы выражений.

**Что изучается**:
- Арифметические выражения
- Логические выражения
- Операторы сравнения
- Комбинированные выражения

**Запуск**:
```bash
datacode examples/ru/02-синтаксис/выражения.dc
```

### 3. `булевы.dc` - Булевы значения
**Описание**: Демонстрирует работу с булевыми значениями.

**Что изучается**:
- Булевы литералы: `true`, `false`
- Логическое отрицание: `!`
- Операторы сравнения
- Проверка на `null`

**Запуск**:
```bash
datacode examples/ru/02-синтаксис/булевы.dc
```

## 🎯 Изучаемые концепции

### Условные конструкции
```dc
if условие {
    // код
}

if условие {
    // код
} else if условие {
    // код
} else {
    // код
}
```

### Операторы сравнения
- `==` - равно
- `!=` - не равно
- `>` - больше
- `<` - меньше
- `>=` - больше или равно
- `<=` - меньше или равно

### Логические операторы
- `!` - логическое отрицание
- `and` - логическое И (если поддерживается)
- `or` - логическое ИЛИ (если поддерживается)
