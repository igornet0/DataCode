# 📦 Типы данных в DataCode

Этот раздел демонстрирует работу с различными типами данных в DataCode, включая преобразование типов, проверку типов и работу с объектами.

## 📋 Содержание

### 1. `функции_преобразования_типов.dc` - Функции преобразования типов
**Описание**: Демонстрирует все функции преобразования типов, доступные в DataCode.

**Что изучается**:
- Преобразование между типами с помощью `int()`, `float()`, `bool()`, `str()`
- Создание массивов с помощью `array()`
- Работа с датами с помощью `date()`
- Форматирование денег с помощью `money()`
- Определение типов с помощью `typeof()`
- Практические примеры преобразования типов

**Запуск**:
```bash
datacode examples/ru/03-типы\ данных/функции_преобразования_типов.dc
```

### 2. `руководство_преобразования_типов.dc` - Руководство по преобразованию типов
**Описание**: Подробное руководство по работе с типами и преобразованиям.

**Что изучается**:
- Базовые типы данных
- Проверка типов с помощью `isinstance()`
- Стратегии преобразования типов
- Работа с различными комбинациями типов

**Запуск**:
```bash
datacode examples/ru/03-типы\ данных/руководство_преобразования_типов.dc
```

### 3. `тип_дата.dc` - Работа с датами
**Описание**: Демонстрирует работу с датами и форматированием.

**Что изучается**:
- Создание дат
- Форматы дат
- Работа со строковыми датами

**Запуск**:
```bash
datacode examples/ru/03-типы\ данных/тип_дата.dc
```

### 4. `объекты.dc` - Работа с объектами
**Описание**: Демонстрирует создание и работу с объектами (словарями) в DataCode.

**Что изучается**:
- Создание объектов с парами ключ-значение
- Доступ к свойствам через точку (`obj.property`)
- Доступ к свойствам через квадратные скобки (`obj['key']`)
- Вложенные объекты
- Объекты с разными типами значений
- Объекты в массивах
- Использование объектов как словарей
- Практические примеры конфигурации

**Запуск**:
```bash
datacode examples/ru/03-типы\ данных/объекты.dc
```

## 🎯 Изучаемые концепции

### Преобразование типов
- **int()**: Преобразование в целое число
- **float()**: Преобразование в число с плавающей точкой
- **bool()**: Преобразование в булево значение
- **str()**: Преобразование в строку
- **array()**: Создание массива из аргументов
- **date()**: Создание даты из строки
- **money()**: Форматирование числа как валюты

### Проверка типов
- **typeof()**: Получение имени типа в виде строки
- **isinstance()**: Проверка, является ли значение экземпляром типа

### Объекты (Словари)
- **Создание**: `{key: value, key2: value2}`
- **Доступ**: `obj.key` или `obj['key']`
- **Вложенные**: Объекты могут содержать другие объекты
- **Смешанные типы**: Значения могут быть любого типа DataCode
- **Строковые ключи**: Используйте кавычки для ключей с пробелами: `{'key name': value}`
