# 🎯 Продвинутые возможности ДатаКод

Этот раздел содержит продвинутые примеры использования языка.

## 📋 Содержание

### 1. `комплексный.dc` - Комплексный пример
**Описание**: Демонстрирует комбинацию различных возможностей языка.

**Что изучается**:
- Комбинация функций, циклов и условий
- Рекурсивные функции
- Сложные вычисления
- Практические примеры использования

**Запуск**:
```bash
datacode examples/ru/04-продвинутые/комплексный.dc
```

### 2. `демо_области_видимости.dc` - Область видимости
**Описание**: Демонстрирует работу с глобальными и локальными переменными.

**Что изучается**:
- Глобальные переменные
- Локальные переменные в функциях
- Переменные в циклах и блоках
- Область видимости в рекурсивных функциях

**Запуск**:
```bash
datacode examples/ru/04-продвинутые/демо_области_видимости.dc
```

### 3. `обработка_ошибок.dc` - Обработка ошибок
**Описание**: Демонстрирует обработку ошибок с помощью блоков try/catch/throw/finally.

**Что изучается**:
- Базовые блоки try/catch
- try/catch с переменной ошибки и без неё
- Блоки try/finally
- Комбинация try/catch/finally
- Использование throw для создания исключений
- Вложенные блоки try/catch
- Обработка ошибок в функциях
- Операции с файлами с обработкой ошибок
- Распространение ошибок

**Запуск**:
```bash
datacode examples/ru/04-продвинутые/обработка_ошибок.dc
```

## 🎯 Изучаемые концепции

### Область видимости переменных
- **Глобальные переменные**: Объявленные на верхнем уровне, доступны везде
- **Локальные переменные**: Объявленные внутри функций, доступны только внутри функции
- **Параметры функций**: Локальные переменные функции
- **Переменные в блоках**: Доступны только внутри блока (if, while, for)

### Обработка ошибок
- **try/catch**: Перехват и обработка исключений
- **try/finally**: Всегда выполнять код очистки
- **throw**: Создание пользовательских исключений
- **Вложенные блоки**: Обработка ошибок на разных уровнях
- **Распространение ошибок**: Ошибки всплывают через вызовы функций

### Композиция функций
Функции можно комбинировать для создания более сложных решений:
- Функции могут вызывать другие функции
- Функции могут быть аргументами других функций
- Рекурсивные функции могут использовать локальные переменные
