# 🎪 Демонстрации ДатаКод

Этот раздел содержит комплексные демонстрации всех возможностей языка.

## 📋 Содержание

### 1. `демонстрация.dc` - Полная демонстрация
**Описание**: Комплексный пример, демонстрирующий все основные возможности языка ДатаКод.

**Что демонстрируется**:
- Переменные и арифметические операции
- Работа со строками
- Условные конструкции (if/else)
- Функции и их использование
- Рекурсивные функции
- Циклы while и for
- Вложенные циклы
- Комплексные вычисления
- Комбинация всех возможностей

**Запуск**:
```bash
datacode examples/ru/06-демонстрации/демонстрация.dc
```
