# 🔄 Циклы в ДатаКод

Этот раздел содержит примеры использования циклов `while` и `for`.

## 📋 Содержание

### 1. `циклы_while.dc` - Циклы while
**Описание**: Демонстрирует использование цикла `while`.

**Что изучается**:
- Синтаксис цикла `while`
- Условия продолжения цикла
- Изменение переменных в цикле
- Условия выхода из цикла

**Запуск**:
```bash
datacode examples/ru/07-циклы/циклы_while.dc
```

### 2. `циклы_for.dc` - Циклы for
**Описание**: Демонстрирует использование цикла `for`.

**Что изучается**:
- Синтаксис цикла `for`: `for (init; condition; increment) { ... }`
- Инициализация переменной цикла
- Условие продолжения
- Инкремент/декремент
- Использование цикла for в функциях

**Запуск**:
```bash
datacode examples/ru/07-циклы/циклы_for.dc
```

### 3. `вложенные_циклы.dc` - Вложенные циклы
**Описание**: Демонстрирует вложенные циклы и их комбинации.

**Что изучается**:
- Вложенные циклы `while`
- Вложенные циклы `for`
- Комбинации циклов и условий
- Практические примеры (таблица умножения)

**Запуск**:
```bash
datacode examples/ru/07-циклы/вложенные_циклы.dc
```

## 🎯 Изучаемые концепции

### Цикл while
```dc
while условие {
    // код, который выполняется пока условие истинно
}
```

### Цикл for
```dc
for i in iter {
    // тело цикла
}
```

**Примеры**:
- `for i in range(1, 11) { ... }`
- `for i in range(-10, 2) { ... }`
- `for (; условие; инкремент) { ... }` - без инициализации

### Важные моменты
1. **Условие выхода**: Убедитесь, что условие цикла когда-нибудь станет ложным
2. **Изменение переменных**: Не забывайте изменять переменные в цикле
3. **Бесконечные циклы**: Будьте осторожны с условиями, которые всегда истинны
