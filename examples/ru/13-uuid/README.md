# UUID в DataCode

Этот раздел демонстрирует работу с встроенным модулем `uuid`: генерация уникальных идентификаторов (v4, v7), разбор строк, работа с байтами, детерминированные UUID (v3, v5) и метаданные.

## 📋 Содержание

### 1. `01-базовое_использование.dc` - Базовое использование
**Описание**: Импорт модуля, генерация UUID v4 и v7, вывод в строку, разбор строки в UUID.

**Что изучается**:
- Импорт модуля `uuid`
- Генерация случайного UUID: `uuid.v4()`, `uuid.random()`
- Генерация время-упорядоченного UUID: `uuid.v7()`, `uuid.new()`
- Преобразование в строку: `uuid.to_string(u)`
- Разбор строки: `uuid.parse(s)` и проверка на `null`
- Определение типа: `typeof(u)`

**Запуск**:
```bash
datacode examples/ru/13-uuid/01-базовое_использование.dc
```

### 2. `02-детерминированные_uuid.dc` - Детерминированные UUID (v3, v5)
**Описание**: Генерация воспроизводимых UUID по namespace и имени с помощью v3 и v5.

**Что изучается**:
- Функции `uuid.v3(namespace, name)` и `uuid.v5(namespace, name)`
- Стандартные namespace: `uuid.DNS`, `uuid.URL`, `uuid.OID`
- Одинаковые namespace и имя дают один и тот же UUID

**Запуск**:
```bash
datacode examples/ru/13-uuid/02-детерминированные_uuid.dc
```

### 3. `03-байты.dc` - Работа с байтовым представлением
**Описание**: Преобразование UUID в массив байтов и обратно.

**Что изучается**:
- `uuid.to_bytes(u)` — получение массива из 16 чисел (0–255)
- `uuid.from_bytes(arr)` — сборка UUID из массива
- Круговая проверка: UUID → bytes → UUID → to_string

**Запуск**:
```bash
datacode examples/ru/13-uuid/03-байты.dc
```

### 4. `04-метаданные.dc` - Метаданные UUID
**Описание**: Получение версии, варианта и временной метки из UUID.

**Что изучается**:
- `uuid.version(u)` — версия UUID (1–7)
- `uuid.variant(u)` — вариант (0=NCS, 1=RFC4122, 2=Microsoft, 3=Future)
- `uuid.timestamp(u)` — Unix-время для v1/v7 или `null` для v4

**Запуск**:
```bash
datacode examples/ru/13-uuid/04-метаданные.dc
```

### 5. `05-практический_пример.dc` - Практический пример
**Описание**: Генерация идентификаторов для записей, вывод и сохранение в структуры данных.

**Что изучается**:
- Типичное использование `uuid.v7()` для уникальных id записей
- Вывод в виде строк и сохранение в массив/объект

**Запуск**:
```bash
datacode examples/ru/13-uuid/05-практический_пример.dc
```

## 🎯 Изучаемые концепции

### Генерация UUID
- **uuid.v4()**, **uuid.random()**: Случайный UUID (версия 4)
- **uuid.v7()**, **uuid.new()**: Время-упорядоченный UUID (версия 7), удобен для индексов

### Разбор и строки
- **uuid.parse(s)**: Строка в формате с дефисами → UUID или `null` при ошибке
- **uuid.to_string(u)**: UUID → строка в стандартном формате

### Байты
- **uuid.to_bytes(u)**: UUID → массив из 16 чисел 0–255
- **uuid.from_bytes(arr)**: Массив из 16 чисел → UUID

### Детерминированные UUID
- **uuid.v3(namespace, name)**: UUID версии 3 (MD5)
- **uuid.v5(namespace, name)**: UUID версии 5 (SHA-1)
- **uuid.DNS**, **uuid.URL**, **uuid.OID**: Стандартные namespace для v3/v5

### Метаданные
- **uuid.version(u)**: Номер версии (1–7)
- **uuid.variant(u)**: Вариант (0–3)
- **uuid.timestamp(u)**: Unix-время для v1/v7, для v4 — `null`
