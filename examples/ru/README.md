# 📚 Примеры ДатаКод

Добро пожаловать в коллекцию примеров для языка программирования ДатаКод!

## 🎯 Структура примеров

Примеры организованы по разделам от простого к сложному:

### 🚀 [01-основы](01-основы/) - Основы языка
Базовые примеры для начала изучения:
- `привет.dc` - Простейший пример
- `переменные.dc` - Работа с переменными
- `арифметика.dc` - Арифметические операции
- `строки.dc` - Работа со строками
- `глобальные_локальные.dc` - Глобальные и локальные переменные

### 🔧 [02-синтаксис](02-синтаксис/) - Синтаксические конструкции
Примеры синтаксиса языка:
- `условия.dc` - Условные конструкции (if/else)
- `выражения.dc` - Сложные выражения
- `булевы.dc` - Булевы значения и логические операции

### 📦 [03-типы данных](03-типы%20данных/) - Типы данных
Работа с различными типами данных:
- `функции_преобразования_типов.dc` - Функции преобразования типов
- `руководство_преобразования_типов.dc` - Руководство по преобразованию типов
- `тип_дата.dc` - Работа с датами
- `объекты.dc` - Работа с объектами (словарями)

### 📦 [05-функции](05-функции/) - Функции
Примеры создания и использования функций:
- `простые_функции.dc` - Простые функции
- `рекурсия.dc` - Рекурсивные функции
- `вложенные_функции.dc` - Вложенные вызовы функций

### 🔄 [07-циклы](07-циклы/) - Циклы
Примеры использования циклов:
- `циклы_while.dc` - Циклы while
- `циклы_for.dc` - Циклы for
- `вложенные_циклы.dc` - Вложенные циклы

### 🎯 [04-продвинутые](04-продвинутые/) - Продвинутые возможности
Продвинутые техники программирования:
- `комплексный.dc` - Комплексный пример
- `демо_области_видимости.dc` - Область видимости переменных
- `обработка_ошибок.dc` - Обработка ошибок с try/catch/throw

### 🎪 [06-демонстрации](06-демонстрации/) - Демонстрации
Комплексные демонстрации всех возможностей:
- `демонстрация.dc` - Полная демонстрация языка

### 📊 [09-создание модели данных](09-создание%20модели%20данных/) - Создание моделей данных
Построение моделей баз данных из CSV файлов:
- `01-операции_с_файлами.dc` - Работа с файлами и директориями
- `02-объединение_таблиц.dc` - Объединение нескольких таблиц
- `03-создание_связей.dc` - Создание связей между таблицами
- `04-загрузка_квартальных_данных.dc` - Загрузка квартальных агрегированных данных
- `05-объединения_таблиц.dc` - Операции объединения таблиц (inner, left, right, full, cross, semi, anti)
- `загрузка_данных_модели.dc` - Полный пример загрузки всех данных и создания базы данных SQLite

### 🎨 [10-графики](10-графики/) - Графики и визуализация
Работа с модулем `plot` для загрузки и отображения изображений, а также построения графиков:
- `01-загрузка-изображения.dc` - Загрузка изображения из файла
- `02-показ-изображения.dc` - Быстрый просмотр изображения
- `03-создание-окна.dc` - Создание окна для отрисовки
- `04-рисование-изображения.dc` - Рисование изображения в окне
- `05-полный-пример.dc` - Комплексный пример работы с модулем plot
- `06-subplots.dc` - Создание сетки subplots
- `07-столбчатая-диаграмма.dc` - Построение столбчатой диаграммы
- `08-тепловая-карта.dc` - Построение тепловой карты
- `09-линейный-график.dc` - Построение линейного графика
- `10-круговая-диаграмма.dc` - Построение круговой диаграммы

### 🧠 [11-mnist-mlp](11-mnist-mlp/) - Пример MNIST MLP
Обучение многослойного перцептрона (MLP) на наборе данных MNIST:
- `mnist_mlp.dc` - Полный пример обучения MLP
- `mnist_model_demo.dc` - Демонстрация модели
- `mnist_mlp_sh.dc` - Пример shell скрипта

### ⚙️ [12-settings_env](12-settings_env/) - Модуль settings_env
Загрузка переменных окружения из .env файлов, приведение типов, префиксы, config и классы-наследники Settings:
- `01-базовое_использование.dc` - load_env, Settings, чтение ключей
- `02-типы_и_приведение.dc` - Приведение к bool, number, string
- `03-префикс_и_config.dc` - env_prefix и load_env с config
- `04-field.dc` - Дескрипторы Field (default, required, min_length, alias)
- `05-класс_config.dc` - Класс-наследник Settings с вложенным конфигом
- `06-практический_пример.dc` - Сводный пример конфигурации приложения

### 🆔 [13-uuid](13-uuid/) - Модуль UUID
Работа с встроенным модулем `uuid`: генерация уникальных идентификаторов (v4, v7), разбор строк, байты, детерминированные UUID (v3, v5) и метаданные:
- `01-базовое_использование.dc` - Импорт, v4/v7, to_string, parse
- `02-детерминированные_uuid.dc` - v3 и v5 с namespace (DNS, URL, OID)
- `03-байты.dc` - to_bytes и from_bytes
- `04-метаданные.dc` - version, variant, timestamp
- `05-практический_пример.dc` - Генерация id для записей

## 🚀 Быстрый старт

### Рекомендуемый порядок изучения

1. **Начните с основ**:
   ```bash
   datacode examples/ru/01-основы/привет.dc
   datacode examples/ru/01-основы/переменные.dc
   ```

2. **Изучите синтаксис**:
   ```bash
   datacode examples/ru/02-синтаксис/условия.dc
   datacode examples/ru/02-синтаксис/выражения.dc
   ```

3. **Поймите типы данных**:
   ```bash
   datacode examples/ru/03-типы\ данных/функции_преобразования_типов.dc
   datacode examples/ru/03-типы\ данных/объекты.dc
   ```

4. **Освойте функции**:
   ```bash
   datacode examples/ru/05-функции/простые_функции.dc
   datacode examples/ru/05-функции/рекурсия.dc
   ```

5. **Изучите циклы**:
   ```bash
   datacode examples/ru/07-циклы/циклы_while.dc
   datacode examples/ru/07-циклы/циклы_for.dc
   ```

6. **Продвинутые техники**:
   ```bash
   datacode examples/ru/04-продвинутые/комплексный.dc
   datacode examples/ru/04-продвинутые/обработка_ошибок.dc
   ```

7. **Создание моделей данных**:
   ```bash
   datacode examples/ru/09-создание\ модели\ данных/05-объединения_таблиц.dc
   ```

8. **Графики и визуализация**:
   ```bash
   datacode examples/ru/10-графики/01-загрузка-изображения.dc
   datacode examples/ru/10-графики/07-столбчатая-диаграмма.dc
   datacode examples/ru/10-графики/09-линейный-график.dc
   datacode examples/ru/10-графики/10-круговая-диаграмма.dc
   datacode examples/ru/10-графики/08-тепловая-карта.dc
   ```

9. **Пример MNIST MLP**:
   ```bash
   datacode examples/ru/11-mnist-mlp/mnist_mlp.dc
   ```

10. **Модуль settings_env**:
   ```bash
   datacode examples/ru/12-settings_env/01-базовое_использование.dc
   datacode examples/ru/12-settings_env/05-класс_config.dc
   ```

11. **Модуль UUID**:
   ```bash
   datacode examples/ru/13-uuid/01-базовое_использование.dc
   datacode examples/ru/13-uuid/05-практический_пример.dc
   ```

12. **Полная демонстрация**:
   ```bash
   datacode examples/ru/06-демонстрации/демонстрация.dc
   ```
